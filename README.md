# DevsarrollaIT
Es una web app que conecta a personas que buscan desarrollar su carrera profesional en el mundo IT con proyectos de interes de los cuales pueden formar parte.

### Design

-  *Figma*


### Frontend

- *Vue*
- *Pinia*
- *Tailwind*
- *Css*
- *Vite*

### Backend

- *Node.js*
- *TypeScript*
- *Express*
- *jsonwebtoken*
- *MongoDB*
- *Mongoose*
